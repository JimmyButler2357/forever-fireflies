01 — Foundations

Colors

Warm cream, dark brown, and a single orange accent. Gold is reserved for "magic" moments — fireflies, celebrations, the Firefly Jar. Never use raw hex in components — pull from the theme.

02 — Foundations

Typography

Two voices: Merriweather serif for the journal (titles, transcripts, prompts); system sans for the app chrome (labels, buttons, metadata). Five named sizes only — never invent in-between values.

When to reach for serif

App title · onboarding headings · transcript body · prompt cards · Firefly Jar title and previews · empty-state messages · birthday picker selected value · child-name input. Everything else is system sans.

07 — States

Empty, loading & error

Every screen handles all three. Never a blank screen, never a spinner alone, never "Error 500" — always plain language and a way forward.

No fireflies yet

Tap the heart on any entry to save it as a Firefly.

08 — Motion

09 — Voice

Tone & language

Warm, encouraging, never clinical. Use the child's name whenever possible. Frame features as emotional outcomes — kept forever, not stored in cloud. No streaks, no guilt, no missed-day counters.
